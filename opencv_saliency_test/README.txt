There are four methods for saliency in opencv_contrib: 3 for images, 1 for video(motion)

To run the program:
./sal_speed face.png option

where the option can be: s or f or b or m, 
in which s,f,b speed is tested sperately: 180~230ms, 175~185ms, 570~590ms
 
for example:
./sal_speed face.png m

check the result dir to observe different saliency result
